#  RobotFrameworkProject

Projet d'automatisation des tests utilisant **Robot Framework** 
avec **SeleniumLibrary** sur l'application SauceLabs.

## Technologies utilisées

| Technologie | Usage |
|-------------|-------|
| Robot Framework | Framework de test |
| SeleniumLibrary | Automatisation UI |
| SauceLabs | Application testée |
| Python 3.x | Langage de base |
| VS Code | IDE |

---

##  Contenu

###  SauceLabsProject
- Connexion valide / invalide
- Navigation dans l'application
- Screenshots automatiques



##  Installation

```bash
pip install robotframework
pip install robotframework-seleniumlibrary
```
